# Changelog

All notable changes to the "HTML5 Templates" extension will be documented in this file.

## [1.2.0] - 2026-03-06
### Added
- **Snippet Logic**: Converted static templates into interactive SnippetStrings.
- **Mirrored Editing**: Updating the Page Title now automatically updates Open Graph, Twitter, and Schema tags.
- **Automatic Variables**: Added `CURRENT_YEAR` and `DATE` logic to the Blog template.

### Changed
- Refactored `extension.js` for better performance.
- Updated Meta Tag standards for 2026 browser requirements.

## [1.0.0] - 2025
- Initial release with basic HTML5 and Blog templates.
