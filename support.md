# Support

If you encounter any issues or have suggestions for new templates, please use the following channels:

## 🐛 Bug Reports
Please report bugs via the **Issues** tab on our [GitHub Repository](https://github.com/deepaksingh1984/seo-ocean/issues). 

When reporting a bug, please include:
- Your VS Code version.
- A brief description of what happened.

## 💡 Feature Requests
Want a specific framework template (like React or Vue)? Open an issue with the tag `enhancement`.

## ✉️ Contact
For other inquiries, you can reach out via email at: `info@binari-software.com`
